Deqode Assignment
=============================================

**Objective** : This Project allows user to upload a file and download it.

API Structure :

1. **Upload File :**     
   Request Type : POST      
   URL : /api/v1/image    
   Body : { pic : MIMETYPE}   

2. **Download File :**     
   Request Type : GET    
   URL : /api/v1/image/:filename
